Bottle et Peewee

Posted on Sam 16 décembre 2017 in web

Dans mes développements, dès que j'ai besoin d'afficher et gérer quelques pages web, je pense automatique à Bottle. Ce mini-framework web (il tient en un fichier) fait le minimum, mais le fait bien ;-).
Peewee est un ORM (object-relational mapping) qui permet de manipuler des bases de données en utilisant des objets Python. Ceci permet de se détacher des problématiques liées aux bases de données (ouverture, requêtes SQL, sauvegarde, fermeture, etc) et d'avoir ainsi un code faiblement couplé à une techno BDD.
